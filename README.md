# rulespec-us

United States federal RuleSpec encodings and source registry metadata.

## Layout

- `statutes/`: federal statute RuleSpec YAML, with tests beside each encoding as `.test.yaml`.
- `regulations/`: federal regulation RuleSpec YAML, with tests beside each encoding as `.test.yaml`.
- `policies/`: federal policy RuleSpec YAML, with tests beside each encoding as `.test.yaml`.
- `sources/`: source registry or manifest metadata when needed.

Do not add singular rule roots, separate parameter/test fixture files, or
generated formula artifacts.
